# Zim.app

Unofficial OsX package for ZIM Desktop Wiki.

http://zim-wiki.org/

Current base version: 0.63

## How to Install

1. Create a Notes file
```bash
cd ~
mkdir Notebooks
touch Notebooks/Notes
```
2. Drag Zim.app to the Applications folder
3. Use the Zim App
